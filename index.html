Instructions:

Select either "Power" or "Comparison" as the operation in the Actions Tab to be performed on the first and second values.
Fill in correctly the necessary values in the Calculation Tab. X and Y values can only be positive integers and the running tape speed is always at least 100 ms fast for each interval between configurations.
Select "Initialize" under the Actions Tab.
You may now run the tape until a configuration reaches the halting state using the "Start" button, or do the same thing configuration by configuration using the "Step" button.
You may pause the running tape using "Pause" or reset the Turing machine completely using "Reset".
The answer is found in the input tape right after the '@'.
